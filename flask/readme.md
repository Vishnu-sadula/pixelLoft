update python3, pip and venv -- sudo apt install python3-pip python3-venv

To run this Application
1. Clone the repo
2. Get into the Folder, update the pip and venv
3. Create virtual env -- python3 -m venv venv
4. Activate venv -- source venv/bin/activate
5. Install the requirements -- python3 -m pip install -r requirements.txt
6. python3 first_flask.py
