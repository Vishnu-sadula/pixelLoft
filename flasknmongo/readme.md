This is the Flask app interacting with MongoDB locally to store basic data.

update python3, pip and venv sudo apt install python3-pip python3-venv

To run this Application

1. Clone the repo
2. Get into the project folder, run python3 -m venv venv
3. RUN source venv/bin/activate -- to activate the virtual environment
4. RUN python3 -m pip install -r requirements.txt -- to install the requirements needed to run the project
5. To access the application, run python3 app.py
